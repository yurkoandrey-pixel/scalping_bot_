# 🤖 Скальпинг Бот для Bybit

Автоматический бот для скальпинга на криптобирже Bybit.
Стратегия: Parabolic SAR с мгновенным Take Profit 0.2%.

## 🚀 Быстрый старт

### 1. Установка Python

Убедись, что установлен Python 3.8+:
```bash
python3 --version
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Настройка

Открой `config.py` и проверь настройки:

```python
# Режим работы
LIVE_TRADING = False  # False = симуляция, True = реальная торговля

# API ключи (уже настроены для Read-Only)
API_KEY = "твой_ключ"
API_SECRET = "твой_секрет"

# Параметры торговли
TP_PERCENT = 0.2  # Take Profit 0.2%
```

### 4. Запуск

```bash
python bot.py
```

## 📊 Как это работает

### Режим симуляции (LIVE_TRADING = False):
1. Бот сканирует ~50 волатильных монет
2. Ищет сигналы разворота SAR
3. Показывает виртуальные входы и выходы
4. Считает статистику (Win Rate, P&L)
5. НЕ торгует реально

### Режим торговли (LIVE_TRADING = True):
1. Всё то же самое
2. НО открывает реальные сделки
3. Выставляет реальный Take Profit
4. Требует API ключ с правами Trade

## ⚙️ Настройки в config.py

### Основные:
- `LIVE_TRADING` — False = симуляция, True = реальная торговля
- `TP_PERCENT` — Take Profit в % (по умолчанию 0.2)
- `SL_PERCENT` — Stop Loss в % (0 = выключен)
- `LEVERAGE` — Плечо (по умолчанию 10)

### Фильтры монет:
- `MIN_VOLATILITY` — Минимальная волатильность % (0.2)
- `MAX_VOLATILITY` — Максимальная волатильность % (10.0)
- `MIN_VOLUME_24H` — Минимальный объём за 24ч ($1,000,000)
- `BLACKLIST` — Список монет для игнорирования

## 🔄 Переход к реальной торговле

1. Получи API ключ с правами **Trade** на Bybit
2. Замени ключи в `config.py`:
   ```python
   API_KEY = "новый_ключ_с_trade"
   API_SECRET = "новый_секрет"
   ```
3. Измени режим:
   ```python
   LIVE_TRADING = True
   ```
4. Запусти бота:
   ```bash
   python bot.py
   ```

## ⚠️ Важно

- Начни с симуляции — убедись что всё работает
- Для реальной торговли используй маленький депозит сначала ($50-100)
- Бот торгует на волатильных альткоинах — это рискованно
- Не оставляй бота без присмотра первое время

## 📈 Стратегия

```
ВХОД:
• SAR развернулся СВЕРХУ → SHORT
• SAR развернулся СНИЗУ → LONG
• Вход мгновенный (на той же свече)

ВЫХОД:
• TP 0.2% — выставляется сразу
• Закрытие автоматическое

МОНЕТЫ:
• Волатильность 0.2% - 10% на свечу
• Объём > $1M за 24ч
• Без BTC, ETH, SOL, XRP
```

## 🛠 Решение проблем

### "Не удалось подключиться к Bybit"
- Проверь интернет
- Проверь правильность API ключей
- Убедись что аккаунт не заблокирован

### "Нет сигналов"
- Подожди — рынок не всегда даёт сигналы
- Проверь что список монет не пустой

### "API Error"
- Проверь права API ключа
- Для торговли нужны права Trade
